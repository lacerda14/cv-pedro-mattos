Calculadora Vento Fresco

Página da web criada para a empresa Vento Fresco que visa a auxilar os clientes a calcularem a quantidade de equipamentos de equipamentos de climatização seriam necessários para seu evento baseado na m². Apesar de ser um dos meus projetos mais rudimentares na área de CSS e HTML, esse projeto é um dos muitos que tenho que usam o JavaScript para fazer a parte funcional da calculadora e da interação com o usuário.
